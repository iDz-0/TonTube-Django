Comment utiliser le site : 
Créer un compte / connectez vous avez les buttons en haut a gauche du menu (toutes les pages)
Reagardez les videos disponible sur la page d'acceuil ou, triez les en fonction de vos abonnements, historique ou tendance
Une fois sur la page d'une video vous pouvez :
 - Liker
 - Commenter
 - Accéder à la chaine qui a poster la vidéo
 - Voir d'autre vidéos asscociées
 
 Sur une page de chaine vous pouvez : Vous abonnez, regardez les autres videos de la chaine
 
 Pour créer votre chaine vous pouvez cliquer sur "my channel" dans le menu en haut a gauche, indiquez le nom de votre chaine ainsi qu'un logo et une bannière (au même format que sur youtube)
 Ensuite utilisez le boutton a droite de la barre de recherche pour uploader votre vidéo (l'importer depuis youtube en faite, en idiquant son url)
